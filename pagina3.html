Lenguajes de Programación

Los microcontroladores pueden ser programados utilizando varios lenguajes de programación. A continuación, se presentan algunos de los más comunes:

Lenguaje Descripción
C Un lenguaje de programación de propósito general que es ampliamente utilizado en la programación de microcontroladores.
Assembly Un lenguaje de bajo nivel que permite un control preciso sobre el hardware del microcontrolador.
Python Utilizado en plataformas como Raspberry Pi, permite una programación más sencilla y rápida.

Ejemplo de Código en C

Control de un semáforo simple con 3 LEDs:


#include <avr/io.h>
#include <util/delay.h>

int main(void) {
    DDRB = 0b00000111;

    while (1) {
        PORTB = 0b00000001; // Rojo
        _delay_ms(3000);

        PORTB = 0b00000010; // Amarillo
        _delay_ms(1000);

        PORTB = 0b00000100; // Verde
        _delay_ms(3000);
    }
    return 0;
}
